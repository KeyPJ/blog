# 五险两金工资计算器 · 使用说明
基于前端实现的**五险两金（养老保险、医疗保险、失业保险、工伤保险、生育保险、住房公积金、企业年金）** 税前税后工资计算工具，支持自定义基数、比例与地区政策，实时计算到手工资与单位支出。  
  
点击左侧或者直接访问:[计算器](/scal.html)  

## 功能特性
- 支持完整五险两金计算逻辑
- 自定义缴费基数、个人/单位缴纳比例
- 自动计算个人所得税（按现行个税起征点与阶梯税率）
- 实时展示：
    - 个人应缴部分（五险两金）
    - 单位应缴部分（五险两金）
    - 应纳税所得额
    - 个税金额
    - 税后实发工资
- 纯前端计算，无数据上传，保护隐私
- 适配不同城市社保政策，可灵活调整参数

## 使用场景
- 个人工资条核对、月度收入预估
- 跳槽薪资对比、offer 薪资换算
- 企业人事/财务岗位快速试算薪资成本
- 开发者学习社保/公积金/个税计算逻辑参考

## 使用方法
1. 输入税前工资（必填项）
2. 设置社保、公积金、企业年金基数（默认可等于税前工资）
3. 调整个人与单位缴纳比例（可按当地最新政策填写）
4. 工具自动计算并展示完整的薪资明细

## 技术说明
- 核心计算：纯前端 vue JavaScript 实现
- 数据安全：仅本地计算，不上传任何个人信息

## 重要提示
- 本工具仅为**估算参考**，实际工资、社保、公积金缴纳金额以单位工资条及当地社保局/公积金管理中心官方政策为准
- 社保基数、缴纳比例、个税政策每年可能调整，使用前请核对最新标准
- 不同行业、企业性质、城市的缴纳规则存在差异，可通过自定义参数适配
